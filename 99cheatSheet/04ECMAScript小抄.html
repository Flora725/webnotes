JavaScript常用对象常用方法·String对象
  • concat
    连接两个或多个字符串，类似 +；str1.concat(str2)
  • indexOf
    某个指定的字符串值在字符串中首次出现的位置；str.indexOf(“b”)
  • lastIndexOf()
    一个指定的字符串值最后出现的位置；str.lastIndexOf(“a”)
  • charAt(index)
    返回指定索引处的字符
  • split
    把一个字符串分割成字符串数组，并截取前面若干项；str.split(“”, 3) str.split(“-”, 4)
  • slice(start,end)
    一个新的字符串。从start开始（包括）到end结束（不包括）为止的所有字符；a.slice(1,3)
  • replace(old,new)
    每次只替换一个，全部替换完要循环
  • trim
    从一个字符串的两端删除空白字符

RegExp对象
  • 基本元字符"."
    表示任一个非换行的字符
  • ()
    表示分组和提高优先级
  • []
    表示一个字符，如 [abc] 匹配a/b/c
  • |
    或（优先级最低）用法：正则表达式 | 正则表达式
  • 次数限定元字符"*"
    紧挨前面的一个字符或一组字符出现 0 次到多次
  • +
    紧挨它前面的字符出现 1 次到多次
  • ？
    紧挨它前面的字符出现 0 次或1次
  • {n}
    紧挨它前面的字符出现n次
  • {n, m}
    紧挨它前面的字符出现n~m次
  • 首尾元字符"ˆ"
    以某字符开头
  • $
    以某字符结尾
  • \d
    digit;代表[0-9]；一个数字
  • \D
    [^0-9]；非数字字符
  • \w
    word;代表[a-zA-Z0-9_]；一个单词字符(所有的数字、字母、下划线)
  • \W
    digit;代表[^a-zA-Z0-9_]；非单词字符
  • \s
    space;代表[\f\r\n\t\v]；不可见字符：空格 / tab / enter
  • \S
    代表[^\f\r\n\t\v]；可见字符
  • 创建正则对象的方法
    var reg=/正则表达式/
    或者var reg=new RegExp(“正则表达式”);

函数调用模式
  • 函数模式
    单独独立调用的就是函数。this关键字指向的是函数的调用者，即调用该方法的对象。在全局上下文中，this表示全局对象。
    function fn(){
    console.log(this);
    }
    fn();
    打印结果：window
  • 方法模式
    方法本身就是函数，但是方法不是单独独立的调用，而是通过一个对象引导调用。this表示引导方法的宿主对象。
    obj.fn = function(){
    console.log(this)
    };
    obj.fn();
    打印结果：Object {}
  • 构造器模式
    用new引导。new是一个运算符，专门用来申请创建对象，创建出来的对象传递给构造函数的this(this指的就是new出来的实例对象)，利用构造函数对其进行初始化。
    ( new Person () ).sayHello()
  • 上下文模式
    ①　call形式：fn.call( obj1, arg1, arg2, arg3,...)
    ②　apply形式：fn.apply( obj1, [arguments] )
    这种调用中的this指向很明确，就是obj1，如果第一个参数是null / underfined，则this指向window。
