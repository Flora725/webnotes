结构标签
  • html
    代表当前书写的是一个html文档
  • head
    存储本页面的一些重要信息，不会显示
  • title
    定义页面标题
  • script
    Javascript代码
  • meta
    元数据（Metadata）是数据的数据信息。通常用于指定网页的描述、关键词、作者、版权等。
  • style
    css样式
  • link
    写在标签中，用来导入css
  • body
    主体内容
  • div
    定义一个区域，与css一起用来布局网页
列表标签
  • ul-li
    无序列表
  • ol/li
    有序列表;PS:li标签要和ul或ol配套使用，但<li>里面可以嵌套其他标签。
文本标签
  • h1...h6
    h1常用于详情页，标记文章的主标题。h1标签在整个页面中是唯一的;h3使用得很频繁，常用于一堆列表的分类标题
  • p
    段落
  • span
    为了突出显示某一小部分内容
  • bytecode
    java字节­码，包­含ja­va虚­拟机能­识别的指令集
  • chartAt(0)
    chartAt(0)
  • equals()
    equals()
表格标签
  • form
    创建表单，并向服务器传输数据;属性有：action(提交路径)，method(get/post)，name(表单名称)
  • input
    规定了用户可以在其中输入数据的控件;属性：type(text、radio、reset、submit、number、password、checkbox、button、image、file)
  • option
    下拉菜单选项
  • textarea
    文本域
