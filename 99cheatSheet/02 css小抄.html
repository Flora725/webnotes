CSS与HTML结合方式
  • 内联样式 行内样式
    将css代码通过style属性来引用
  • 内部样式表
    所有CSS设置都放在style标签中，style放在title下面
  • 外部样式表
    通过link引用 .css文件
基本选择器
  • 标签选择器
    p { } ;直接写标签名
  • 类选择器
    .class { } ;类可多次使用
  • id选择器
    #id { } ;有唯一性
复合选择器
  • 后代选择器
    .list li { } ;选定所有后代
  • 子代选择器
    .list>li { } ;只选子代
  • 并集选择器
    h3,p { } ;所有的都放一起，以逗号相隔
  • 交集选择器
    p.class ;先写标签名，再写类名
  • 通配符*选择器
    选中所有的标签 ;功能强大，效率低下

文本

    颜色 ;取值可以是具体的英文单词;十六进制表示;rgb()表示法

    规定块级元素中的行内元素（文本、图片）的水平对齐方式
  • text-indent
    em/px 定义块级元素中第一个内容行的缩进
  • text-decoration
    underline/line-through
