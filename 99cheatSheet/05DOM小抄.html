节点类型
  • 元素节点
    Type: 1;
    Value: null;
    Name: 大写的标签名
  • 属性节点
    Type: 2;
    Value: 属性值;
    Name: 属性名
  • 文本节点
    Type: 3;
    Value: 文本;
    Name: #text
  • 注释节点
    Type: 8;
    Value: 注释的内容;
    Name: #comment
获取元素·直接获取
  • getElementById
    通过id获取一个元素
  • getElementsByClassName
    通过类名获取一批元素
  • getElementsByTagName
    通过标签名获取一批元素
  • children
    选择子代获取一批元素
  • querySelector
    获取文档中与指定选择器或选择器组匹配的第一个元素
  • querySelectorAll
    获取与指定的选择器组匹配的文档中的一批元素
通过关系获取
  • parentNode
    获取父元素
  • childNodes
    获取所有的子节点 children
  • firstChild
    获取第一个子节点 firstElementChild
  • lastChild
    获取最后一个子节点 lastElementChild
  • previousSibling
    获取上一个兄弟节点 previousElementSibling
  • nextSibling
    获取下一个兄弟节点 nextElementSibling
节点属性操作
  • 获取属性
    object.getAttribute(属性名)
  • 设置属性
    Object.setAttribute(属性名，值)
  • 移除属性
    Object.removeAttribute(属性名)
  • 注意
    setAttribute 并不能改变文章的源码。这种 “表里不一” 的现象源自 DOM 的工作模式：先加载文档的静态内容，再动态刷新，动态刷新不影响文档的静态内容。
动态创建标记
  • 创建标记
    document.createElement
  • document.createTextNode
  • ele.cloneNode(true | false)
  • 插入节点
    fa.appendChild(ele)
  • fa.insertBefore(a,b)
  • 删除标记
    fa.removeChild(ele)
event 对象常用属性
  • type
    事件的类型
  • target
    该事件被传送到的对象
  • bubbles
    是否冒泡
  • screenX
    光标相对于该屏幕的水平位置
  • screenY
    光标相对于该屏幕的垂直位置
  • clientX
    光标相对于该当前窗口的水平位置
  • clientY
    光标相对于该当前窗口的垂直位置
事件类型
  • 鼠标事件
    onmouseover
  • onmouseout
  • onclick
  • ondbclick (适用于 window, imgae)
  • 加载事件
