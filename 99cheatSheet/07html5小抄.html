语义化标签
  • <nav>
    定义导航链接的部分
  • <heade>
    作为文档或文章的页眉
  • <footer>
    文档中的页脚部分
  • <main>
    规定文档的主要内容,在文档中独一无二
  • <article>
    定义外部的内容。比如来自一个外部的新闻提供者的一篇新的文章，或者来自 blog 的文本，或者是来自论坛的文本。亦或是来自其他外部源内容。
  • <aside>
    在<article>中定义 article 以外的内容。aside 的内容应该与 article 的内容相关。
  • <section>
    定义文档中的节（section、区段）。
  • <audio>
    定义声音，比如音乐或其他音频流。
  • <canvas>
    定义图形，比如图表和其他图像
  • <embed>
    定义嵌入的内容，比如插件。
  • <datalist>
    定义可选数据的列表。与 input 元素配合使用
  • <details>
    定义元素的细节，用户可进行查看，或通过点击进行隐藏
  • <progress>
    定义运行中的进度（进程）
表单元素
  • <datalist>
    定义选项列表。
  • <keygen>
    规定用于表单的密钥对生成器字段。
  • <output>
    定义不同类型的输出，比如脚本的输出。
  • <meter>
    定义已知范围或分数值内的标量测量。
表单属性
  • placeholder
    提示内容
  • form
    规定输入域所属的一个或多个表单。注：form 属性必须引用所属表单的 id
  • novalidate
    规定在提交表单时不应该验证 form 或 input 域。
  • autofocus
    规定在页面加载时，域自动地获得焦点。
  • required
    输入域不能为空
  • pattern
    正则验证
  • autocomplete
    规定 form 或 input 域应该拥有自动完成功能。
input的type属性
  • color
    拾色器
  • date
    date 控件
  • email
    用于 e-mail 地址的字段
  • tel
    用于输入电话号码的字段
  • time
    用于输入时间的控件（不带时区）
  • range
    滑动条
  • month
    month 和 year 控件（不带时区）
  • url
    输入 URL 的字段
  • search
    输入搜索字符串的文本字段
  • datetime-local
    date 和 time 控件（包括年、月、日、时、分、秒、几分之一秒，不带时区）
  • file
    选择文件
  • week
    week 和 year 控件（不带时区）
  • calendar
    日历
  • number
    输入数字的字段
